===Redis
基本命令：
set
get

批量设置和批量获取：
mset
mget

查看所有key
keys *

HASH
HMSET key field1 value [...]
HGET key field1

append name "Stephen-Curry[30]"
append age 30 注：若不存在这个key，则默认添加这个key

自增和自减
incr 注：自增1
decr 注：自减1
incrby key 10 注：key自增10
decrby key 10 注：key自减10

查看是否存在及倒计时
EXISTS key
EXPIRE key 10 注：10s后注销
ttl key 注：查看倒计时

type key 判断类型
strlen key 查看长度

GETRANGE key start end 注：对key字符串截取
GETRANGE key 0 -1 注：截取所有

replace
SETRANGE key offset value 注：相当于replace
127.0.0.1:6379> set key1 abcdefg
OK
127.0.0.1:6379> get key1
"abcdefg"
127.0.0.1:6379> setrange key1 1 23
(integer) 7
127.0.0.1:6379> get key1
"a23defg"

根据存在和不存在来设置key
setex 若存在设置（需要设置过期时间）
setnx 若不存在则设置（在分布式锁中常常使用）

msetnx 批量设置不存在的数据（注意：没有msetex方式）
